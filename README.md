## Prerequesites

- [yarn](https://yarnpkg.com)

## Getting Started

```bash
nvm install `cat .nvmrc | head -n 1`
nvm use

npm install --global yarn

yarn install --emoji
yarn run test
yarn run coverage
yarn run audit
```
